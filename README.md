# SPF-for-kids

**Формализованная модель контура изменения мастерства детей**

## О проекте

Этот репозиторий содержит документацию по SPF-pack (Second Principles Framework) — системной модели развития мастерства у детей в возрасте 3-12 лет.

## Содержание

- [SPF-pack v1.1](SPF-pack-v1.1.md) — полная формализованная модель контура изменения мастерства
- [Kids Learning Pack](kids-learning-pack/) — готовая к внедрению спецификация для развития фундаментального интеллекта (возраст 4–7 лет)

## Основные концепции

**SPF-pack** — это управляемая адаптивная система, которая включает:

- Ребёнка как носителя изменяемого состояния мастерства
- Взрослого как агента конфигурации условий
- Среду и повторяющиеся эпизоды практики
- Механизмы обратной связи
- Фиксацию наблюдений и корректировку параметров

## Область применимости

- **Возраст:** 3–12 лет
- **Контексты:** дом, школа/сад, кружки
- **Горизонт наблюдения:** ≥ 2–4 недели повторяемых эпизодов

⚠️ **Важно:** Не применяется для клинической диагностики. При устойчивом страдании обязательна консультация специалиста.

## Ключевые характеристики мастерства

Модель фокусируется на развитии:

- Агентности
- Саморегуляции
- Переноса навыков
- Устойчивости
- Удержания цели
- Качества объяснения
- Работы с ошибкой

## Механизм изменения состояния

Переход состояния мастерства возможен при соблюдении условий:

1. **Повторяемость** — наличие серии эпизодов
2. **Вариативность** — изменяемость контекста для формирования переноса
3. **Обратная связь** — сигнал о соответствии действия критерию
4. **Допустимая фрустрация** — сложность немного выше текущего состояния
5. **Плотность** — интервалы между эпизодами не разрушают когнитивную связность

## Применение

Модель пригодна для:

- Построения методик обучения
- Разработки ИИ-ассистентов для образования
- Построения траекторий развития
- Аудита педагогической практики
