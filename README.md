# SPF-for-kids

**Системная архитектура развития мастерства детей (нулевые, первые и вторые принципы)**

## О проекте

Этот репозиторий содержит полную спецификацию **SPF (Second Principles Framework)** для детского развития — фундаментально обоснованную систему изменения состояния мастерства у детей в реальной жизни.

## Ключевые концепции

### Иерархия
```
ZP (6 нулевых принципов: Axiomaticity, Structure, Multiscale, Optimization, Probability, Computational-feasibility)
  ↓
FPF (Первые принципы: как моделировать системы и эпистемы)
  ↓

  ↓
```

### Что за система SPF-for-kids?

**Это управляемый контур изменения мастерства** (НЕ состояния ребёнка в целом), который включает:
- Ребёнка (активный агент, инициирует)
- Взрослого (конфигуратор условий, слушает больше чем говорит)
- Среду (1–5 эпизодов в неделю, каждый 5–15 мин)
- Обратную связь (эксперимент, не оценка)
- Логирование и наблюдение (видит ли перенос в жизнь)

### 7 характеристик мастерства
(НЕ личности ребёнка, а именно мастерства в работе с различениями)

1. **Агентность** — инициирует, не дожидаясь инструкций
2. **Саморегуляция** — корректирует по результату
3. **Перенос** — применяет в новых ситуациях (это главный метрик)
4. **Устойчивость** — стойко к неудачам (в рамках занятия)
5. **Удержание цели** — помнит, зачем нужно
6. **Качество объяснения** — может рассказать другому
7. **Работа с ошибкой** — анализирует, не стыдится
